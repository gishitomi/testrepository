家計簿アプリ(iOS版)を作成しました

9月上旬ごろにiOS用の家計簿アプリを作成しましたので、その情報共有をさせていただきます。

■アプリの概要について
　家計簿としての必要最低限の機能を持たせています。
　・支出、収入の記録
　・月毎に支出額を参照
　・金額のカテゴライズ機能（食費、雑費など）

■アプリの特徴
　・アプリ内のDBに情報が記録されるので、
　　オフラインでも利用可能
　・会員登録不要（ログイン等する必要なし）

■使用技術
　・フロントエンド(プライバシーポリシーにて使用)
　　HTML
　　CSS
　・バックエンド
　　Swift 5.7（フレームワークは使用していません）
　・データベース
　　SQLite 3.31.1
　その他
　　Git 2.21.0/Github
　　Xcode 14.0.1

Swiftの勉強という目的と、個人的に使いやすい家計簿アプリを
作ってみたいなと思い、作成しました。

  https://apps.apple.com/app/id1644662833
